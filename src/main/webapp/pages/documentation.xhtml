SFT

Documentación

El Sistema de Flujo de Trabajo (SFT) es una aplicación para la gestión del trabajo basado en SCRUM.

Arquitectura


              
 
            Plataforma Java.
            Jakarta EE
            Eclipse Microprofile
            Jmoordb-core
            Base Datos: MongoDB
